# Python Mini Projects

A collection of 5 trending Python mini projects, ready to run and learn from.

## Projects Included
1. **Password Generator** - Generates secure passwords.
2. **QR Code Creator** - Creates QR codes from text or URLs.
3. **Weather API Client** - Fetches current weather using a public API.
4. **CLI Todo App** - Simple command-line task manager.
5. **Mini Web Scraper** - Scrapes links from any website.

## Installation
```bash
git clone https://github.com/german-boop/python-mini-projects.git
cd python-mini-projects
pip install -r requirements.txt
